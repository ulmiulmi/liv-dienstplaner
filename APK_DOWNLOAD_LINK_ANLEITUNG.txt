ULMIPOINT APK-Download-Link Update

Dieses Paket macht zwei Dinge:

1. GitHub Actions veröffentlicht nach dem Build automatisch eine Release-Datei:
   https://github.com/ulmiulmi/liv-dienstplaner/releases/latest/download/ulmipoint-wunschportal.apk

2. wunschportal.html zeigt oben einen Button:
   Android-App herunterladen

Hochladen nach GitHub:
- .github/workflows/build-wunschportal-android.yml ersetzen
- wunschportal.html ersetzen

Danach:
- GitHub -> Aktionen -> Build Wunschportal Android -> Workflow ausführen
- Nach erfolgreichem Lauf ist die APK über den Button im Wunschportal downloadbar.

Wichtig:
Beim ersten Mal gibt es den Download-Link erst, nachdem der Workflow einmal erfolgreich gelaufen ist.
